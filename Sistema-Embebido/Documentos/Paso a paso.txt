Lista de pasos a seguir
1- Comprar todos los materiales.
2- Comenzar a conectar el arduino con el protoboard, la bateria 9v y el LED.
3- Programar el arduino para encender el led 5seg y apagarlo por otros 5seg y asi sucesivamente.
4- Conectar los sensores opticos infrarrojos al arduino.
5- Programar el arduino para hacer detectar una linea en la superficie con los sensores. Mostrar valores por pantalla.
6- Conectar el sensor ultrasonico al arduino.
7- Programar el arduino para calcular objetos segun la distancia con el sensor ultrasonico. Mostrar valores por pantalla.
8- Conectar los motores junto con el puente H al arduino.
9- Programar el arduino para mover las ruedas juntas, luego 5seg la rueda derecha y luego 5seg la rueda izquierda y asi sucesivamente.
10- Recopilar todas las mini-entregas y lograr que el auto siga una linea en el piso con curvas y direcciones.
11- Programar la logica para resolver el laberinto.
12- Programar la logica para aprender el laberinto (Si es posible).

Aclaracion: Todos los pasos contienen su serie de casos de prueba. Una vez que comprobamos que dicha mini-entrega funciona correctamente, recien ahi pasaremos al siguiente paso.
